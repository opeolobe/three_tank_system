# Simulation of Faults in a Three-tank System

## Overview
This repository contains a simulation framework for modeling and analyzing faults in a three-tank system. The project is developed to support research in fault detection and diagnosis (FDD) using model-based techniques.

The system includes dynamic simulation and fault injection.

---

## Objectives
- Simulate the dynamics of a three-tank system
- Analyze the system behavior under different types of inputs in fault-free mode
- Introduce different fault scenarios (multiplicative, sensor, and actuator)
- Analyze system behavior under faulty conditions
---

## Features
- Three-tank dynamic system model
- Fault injection mechanisms
